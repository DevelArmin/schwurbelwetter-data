# HYRAS-Daten

Dieses öffentliche Repository enthält ausschließlich datenschutzreduzierte HYRAS-Zeitreihen aus täglichen TASMAX-Werten sowie einer kompakten TAS-Monatsklimatologie, auswählbaren Monatswerten der verfügbaren Jahre und den abgeschlossenen Monatswerten des aktuellen Jahres.

- Dateinamen: geräteunabhängige, deterministisch erzeugte pseudonyme Ortskennungen
- Keine Ortsnamen
- Keine angeforderten Koordinaten
- Keine Rasterkoordinaten oder Rasterindizes

Die private Zuordnung wird nicht in dieses Repository übertragen.
